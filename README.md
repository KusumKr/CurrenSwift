# CurrenSwift
<br>

<h3>Welcome to CurrenSwift,a Currency Converter repository!</h3>
<br>
<h1>Overview</h1>
<br>
The Currency Converter is a Javascript web application that allows users to convert between different currencies based on the latest exchange rates quickly and efficiently.
<br>
It fetches real-time data fetched from a reliable API to ensure accurate conversions.
<br>

<h1>Features</h1>
<br>
Currency Conversion: Convert from one currency to another.
<br>
Real-time Exchange Rates: Uses up-to-date exchange rates fetched from an API.
<br>
Responsive Design: Designed to work well on desktop and mobile devices.
<br>

<h1>Technologies Used</h1>
<br>
HTML:  Structure of the web page.
<br>
CSS:  Styling and layout.
<br>
JavaScript:  Logic for fetching exchange rates and performing conversions.
<br>
API:  Uses ExchangeRatesAPI to fetch exchange rates.
<br>

<h1>Usage</h1>
<br>
Enter the amount you want to convert and select the currencies.
<br>
Click the Get exchange rate button to see the converted amount.
<br>
